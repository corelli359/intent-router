目录说明：

- `v3/`
  当前生效的主文档目录，只保留最新版本。

- `v2/`
  既有方案、评审、调研、回归和历史设计文档。

- `archive/v3-history/`
  从 `v3/` 归档出的历史版本、阶段性方案和被后续版本替代的协议稿。

- `archive/legacy-notes/`
  历史执行记录、评审和阶段性修复计划。

- `archive/reference-notes/`
  历史设计稿、调研借鉴、汇报材料和参考样例。

- `examples/`
  示例代码。
