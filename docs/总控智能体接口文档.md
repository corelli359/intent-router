

\# 总控智能体接口文档



\## 文档说明



本文档包含总控智能体对外接口、总控调用子智能体接口。



\* \* \*



\## 接口 1：总控智能体规划执行接口



\*   \*\*接口路径\*\*：`Oneagent/process`

\*   \*\*功能说明\*\*：应用调用智能总控，进行任务规划与执行



\### 请求参数



表格



| 字段名 | 字段含义 | 类型 | 是否必填 | 示例值 |

| --- | --- | --- | --- | --- |

| appId | 应用 ID | String | True |  |

| userId | 客户 id | String | True |  |

| sessionId | 总会话 id | String | True |  |

| userinput | 用户输入 | String | True | 先查询再转账 |

| files | 输入文件 | Array | True | \[] |

| stream | 流式响应 | Boolean | True | true |

| variables | 子智能体变量 | Object | True | {} |



\* \* \*



\### 补充说明：sessionId、taskId、chatId 对应关系



sessionId 为用户的一次对话框内对话。对话框内每次新意图分解执行视为一次 task。每个 task 中可涉及任意数量的子智能体。在该次任务中与子智能体的全部对话使用同一个 chatId。 表格



| sessionId | taskId1 | chatId |

| --- | --- | --- |

|  |  | chatId |

|  | taskId2 | chatId |



\* \* \*



\### 响应参数



\#### Event: start



json



```

{

&#x20; "Event": "start",

&#x20; "Data": {

&#x20;   "sessionId": "xx",

&#x20;   "taskId": "xx"

&#x20; }

}



```



\#### Event: IntentRecognition



json



```

{

&#x20; "Event": "IntentRecognition",

&#x20; "Data": {

&#x20;   "thinking": "意图改写结果：xx，意图识别L0，L1，L2，L3，L4，L5。 意图识别结果1，转账，2 查询余额。对应智能体输入：1用户要转账， 2，用户要查询余额"

&#x20; }

}



```



\#### Event: end



json



```

{

&#x20; "Event": "end",

&#x20; "Data": {

&#x20;   "status": "wait",           //总控智能体状态

&#x20;   "currentPlans": \[           //总控智能体计划列表及当前所有子智能体状态

&#x20;     {

&#x20;       "input": "用户要转账",    //智能体用户输入

&#x20;       "intent": "转账",

&#x20;       "agentName": "转账智能体",

&#x20;       "type": "twice",

&#x20;       "status": "processing",

&#x20;       "chatId": "xx",

&#x20;       "variables": {}           //智能体输入变量

&#x20;     },

&#x20;     {

&#x20;       "input": "用户要查询余额",    //智能体用户输入

&#x20;       "intent": "查询余额",

&#x20;       "agentName": "查询余额智能体",

&#x20;       "type": "once",

&#x20;       "status": "pending",

&#x20;       "chatId": "",

&#x20;       "variables": {}           //智能体输入变量

&#x20;     }

&#x20;   ],

&#x20;   "data": {                //计划当前子智能体返回

&#x20;     "output": "请输入转账金额和转账目标卡号"

&#x20;   },

&#x20;   "sessionId": ""

&#x20; }

}



```

\## 接口 2：总控调用子智能体接口



\*   \*\*接口路径\*\*：`/use\_tool`

\*   \*\*功能说明\*\*：总控自主调用子智能体完成任务



\### 请求参数



表格



| 字段名 | 字段含义 | 类型 | 是否必填 | 示例值 |

| --- | --- | --- | --- | --- |

| userId | 客户 id | String | True |  |

| chatId | 子智能体会话 id | String | True |  |

| userinput | 用户输入 | String | True | 先转账再查询 |

| files | 输入文件 | Array | True | \[] |

| stream | 流式响应 | Boolean | True | true |

| variables | 子智能体变量 | Object | True | {} |



\* \* \*



\### 响应参数



\#### Event: start



json



```

{

&#x20; "Event": "start",

&#x20; "Data": {

&#x20;   "chatId": "xx"

&#x20; }

}



```



\#### Event: end



json



```

{

&#x20; "Event": "end",

&#x20; "Data": {

&#x20;   "ishandover": false,      //根据ishandover状态判断子智能体完成情况。1，返回false则代表当前要素缺失，需要补充要素后继续进行。2，返回true并且data中无数据代表该任务处理完成。3，返回true并且data中有数据代表子智能体因意图改写等原因退出，本次任务完成。

&#x20;   "data": "请输入转账金额"

&#x20; }

}



```
