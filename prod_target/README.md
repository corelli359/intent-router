# prod_target

`prod_target/` stores generated deployment-ready frontend artifacts for target clusters.

Current intended contents are generated by `scripts/build_prod_target.sh`:

- `chat-web/`: built standalone chat frontend
- `admin-web/`: built standalone admin frontend
- `k8s/intent/`: rendered target-cluster manifests with host/path parameters

This directory is treated as a local build artifact area. Generated files stay out of git by default.
